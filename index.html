Rem

Rem is a unit of measurment in css; it stants for root em. Unlike em which checks for parent element, rem checks for root element; which is the HTML element.

Percentages:Height and Width

Used to size non-text html elements relative to their parent(also known as a container). Often used to size box-model values like width and height, padding, border, and margins. Be carefull when using 100% for contents may overflow its parents container

Media Queries

CSS uses media queries to adapt content to different screen sizes. CSS detets the the current screen size and apply different CSS styles depending on the width of the screen
